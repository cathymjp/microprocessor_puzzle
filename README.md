# BounceBall
Number puzzle game using EBIMU sensors

## Environment
- 2020.1 ~ 2020.12 (12 months)
- Python3
- pygame
- Microprocessor


## Tools
- EBIMU-9DOFV4 (3-axis gyroscope, 3-axis acceleration sensor, 3-axis accelerometer sensor)
  - pitch, yaw, roll
  - x, y, z
- USB2UART
- serial 
  - Baudrate: 115200
- myCortex-STM32F4
- Comport Master (Program)
